# CareSync 🏥

CareSync is a responsive, front-end clinic appointment booking website. It lets patients browse services and doctors, register/login, and submit appointment requests through a clean, modern interface.

> Built during Nesote Technologies company workshop - "Training on Web-Development with AI" on front-end web development (April 2026).
## 

## Features

* **Home page** with hero section, clinic hours, services overview, and doctor listings
* **Appointment booking form** (patient name, email, phone, date, time, doctor selection, reason for visit)
* **Login \& Registration pages** with client-side validation (email format, password match, required fields)
* **Welcome modal** on the homepage with an auto-closing countdown timer
* **Responsive navigation** with a mobile hamburger menu
* **Interactive doctor cards** with a subtle 3D tilt effect on hover
* Respects `prefers-reduced-motion` for accessibility
* Simple, consistent pink/rose themed UI across all pages
